# Blog Post Generator

Generate SEO-optimized blog posts from outlines and research notes. This tool helps convert brainstorming ideas, research data, and topics into fully formatted, SEO-optimized articles.

## Features

- Outline to full article generation
- SEO optimization (relevant keywords, meta data)
- Topic research analysis
- Multilingual support
- Customizable templates
- CLI interface
- Lexical analysis
- Auto headings and meta tags

## Installation

```bash
git clone https://github.com/Retsumdk/blog-post-generator.git
cd blog-post-generator
npm install
```

## Usage

```bash
# Generate a blog post from an outline
blog-post generate --outline "My Blog Outline" --title "My SEO Blog Title"

# Generate with research notes
blog-post generate --outline @my-research.md --title "Research Blog"

# See full options
blog-post generate --help
```

## Development

```bash
npm install
npm run build
npm test
```
